# 0.2.0

> Shipped 2026-06-04.

The physics-detector release: seven more forces bring the catalog to **33**, matter can now be *created*
and *aged out* (the new source/sink class `[S]`), and the Lab becomes a conformance instrument — fire a
known particle into a force and watch every invariant pass frame-by-frame.

> Draft release notes — paste as the GitHub release body when `v0.2.0` is cut. This file does not itself
> bump versions or tag.

## Added

- **Seven more forces (33 total), spanning every input class.** `memory` (a worn-path occupancy field,
  `[C]`) and `pigment` (conserved colour transport, `[E]`); `pressure` (SPH density relaxation —
  incompressible even-fill, `[B]`); `link` (a Verlet distance constraint — ropes, cloth, soft structures,
  `[B]`); `hunt` (two-species predator/prey pursuit, `[B]`); `morph` (matter assembles into a
  mark/chart/logo — never words; the new shape-assignment class `[D]`); and `spawn` (a budgeted source
  atom that *creates* matter, the new class `[S]`).
- **The source system (class `[S]`).** A `source(b, env)` hook on the `Force` contract — run once per body
  per frame, the dual of `modify` — plus an integrator source pass and an aging/despawn **sink** for
  mortal matter. Sources are budgeted by a per-particle lifespan and a hard pool ceiling, so they can't
  grow the field without bound. Adds the `fountain` preset (a continuous upward jet arcing home under
  gravity).
- **Two more render modes (six total)** — `metaballs` (a liquid iso-surface traced by marching squares)
  and `voronoi` (shattered-glass nearest-neighbour cells), alongside `dots` · `trails` · `links` ·
  `streamlines`.
- **Physics conformance framework** (`conformance/`) — `runScenario` + a declarative `EXPERIMENTS`
  catalog of per-force invariants and exact checks, shared by the test suite and the Lab. Fire a particle
  into a force, verify it reacts as the math predicts.
- **The Lab is a physics detector** — fire known particles into a force and watch the track, the field,
  and related particles resolve each conformance check frame-by-frame. Numeric tuning + presets,
  multi-particle firing, once/loop/unlocked playback, a timeline with a per-particle **speed waveform**
  and a MATCH marker at every test's pass-frame, a parameter-sweep plot, and actionable save (Export JSON
  / Copy report). Handles class-`[S]` sources that start with no test particle.
- **Force-aware Lab controls.** The TUNE & REFIRE panel is driven by each force's catalog attributes — it
  shows only the knobs that matter (shear exposes flow angle; `vortex`/`charge`/`resonate` expose spin;
  class-`[S]` sources show strength + angle), each with its symbol (S, d, σ, θ°), units, a live formula
  line, and a default-value tick on the track. Named quick-pick value bands under each control.
- **Composition + condition experiments** — `COMPOSITE_EXPERIMENTS` verifies that forces compose
  (`attract repel` cancel; `attract vortex` sums to a spiral) and gate on conditions (`data-when` runs
  through the real condition registry).
- **Closed-loop concepts on the Field Manual** — **material typography** (one density, `--d`, drives every
  type axis at once: weight, optical size, tracking, bloom, colour) and a **self-laying-out page**
  (`data-move="layout"` elements find equilibrium via anchor + mutual repulsion + density pressure, and
  re-settle on resize).
- **Conserved attention** (§2.4) — one finite strength budget across the page; engaging a body pulls force
  off the others. Opt-in via `FieldOptions.attention` / `FieldHandle.setAttention` / `<forces-field attention>`.
- **Cross-boundary causality** — density spills from a saturated body to its neighbours (`--lit` +
  `field:lit`/`field:dim` events). Opt-in via `causality`.
- **Developer portal** (`/docs`) — getting started, concepts, framework guides, a catalog-driven API
  reference, recipes, and performance/accessibility notes. Plus `docs/forces-tests.md`, `BACKLOG.md`, and
  release engineering (CI, `CONTRIBUTING.md`, `RELEASING.md`, `SECURITY.md`, issue/PR templates).

## Changed

- **The site front door** — the Field Manual is now the home page (`/reference` redirects to `/`);
  client-side navigation keeps the field running continuously across pages.
- **Performance** — range-culled the integrator body-force loop (~2× at scale), removed all `shadowBlur`
  from the render path, and cached the per-frame `scrollHeight` read.

## Fixed

- **`vortex` whirlpools instead of flinging matter out.** Raised the inward bias `0.12 → 0.6` so particles
  orbit and spiral in rather than drifting outward.
- **Conformance: drag check is now velocity-relative.** Generalized drag's no-redirection check; the
  emitter scenario fires from the nozzle so the demo shows the jet.
- **Docs: restored `forces-formulas.md`** after a LaTeX-corruption revert.
